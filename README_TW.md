# Nuxt Frontend Template

[English](README.md) | 中文

## 使用

```shell
git clome https://github.com/Wuchieh/nuxt-frontend-template.git
cd nuxt-frontend-template
pnpm i
pnpm dev
```

打包

```shell
pnpm build
```
